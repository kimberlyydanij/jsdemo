식별자(identifier) : 프로그램 구현에 필요한 이름을 정의 - 변수명, 함수명, 클래스명 등 카멜 표기법(camel notation)을 사용한다. 변수(variable) : 데이터를 저장하기 위한 메모리 공간 클래스명 : 파스칼 표기법 (pascal notation) 사용 let num = 3 + 5; 키워드, 예약어(keyword) - 프로그램에서 의미를 부여해서 정의 해 놓은 단어 let, var, const 연산자(Operator) : 연산을 하기 위해서 사용되는 기호(산술, 비교, 논리, 조건, 대입) 피연산자(operand) : 연산에 사용되는 데이터 자료형(Data type) : 데이터의 종류 1) Primitive Type : 숫자, 문자, 논리, undefined, null, symbol 2) Reperence Type : 배열[10,20], 객체{ko:90,en:80}, function 호이스팅(Hoisting) : 변수 선언을 최 상위로 끌어올리는 작업
